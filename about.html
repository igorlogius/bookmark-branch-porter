Import or export a branch of the bookmark tree as a JSON or HTML file Short Demo Video: https://github.com/igorlogius/bookmark-branch-porter/assets/67047467/b7d1a2e6-1a01-444e-85c1-52332039dba0 Usage:
  1. click on a bookmark folder
  2. in the "Bookmark Branch Porter" context menu select the export/import options
  3. to remove/hide the context, you can disable the addon
