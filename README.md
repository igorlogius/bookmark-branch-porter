Allows to import or export a branch of the bookmark tree as json or html 

<b>Usage:</b>
<ol>
	<li>install the add-on </li>
	<li>click toolbar icon</li>
	<li>click the import or export button in the popup</li>
	<li>in the import dialog select a folder the file you want to import from </li>
	<li>in the export dialog click export to export all bookmarks otherwise choose a folder beforehand</li>
</ol>
